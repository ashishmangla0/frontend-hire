# Local Instructions

In case of any issues, contact us on [Discord](https://discord.gg/DWAVqksVtx)

## First Time Setup

If you do not have pnpm installed, run the following command:

```sh
npm install -g pnpm
```

Then run the following commands:

```sh
git clone https://github.com/Frontend-Hire/fh-next.git
cd fh-next
pnpm install
cd style-active-route
pnpm install
```

Run the dev server and open the respective dev URL as shown by Vite in your browser.

```sh
pnpm dev
```

In another terminal, run the following command to run the tests:

```sh
pnpm vitest
```

## Already Cloned?

Navigate to the `fh-next` folder and run the following commands:

```sh
git pull

```

Run the dev server and open the respective dev URL as shown by Vite in your browser.

```sh
pnpm dev
```

In another terminal, run the following command to run the tests:

```sh
pnpm vitest
```
