# Local Instructions

In case of any issues, contact us on [Discord](https://discord.gg/DWAVqksVtx)

## First Time Setup

If you do not have pnpm installed, run the following command:

```sh
npm install -g pnpm
```

Then run the following commands:

```sh
git clone https://github.com/Frontend-Hire/fh-dsa.git
cd fh-dsa
pnpm install
cd linked-list-implementation
pnpm vitest index.test.js
```

## Already Cloned?

Navigate to the `fh-dsa` folder and run the following commands:

```sh
git pull
pnpm install
cd linked-list-implementation
pnpm vitest index.test.js
```
